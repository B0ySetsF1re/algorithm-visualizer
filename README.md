# Algorithm Visualizer

Interactive visualizer for sorting and pathfinding algorithms.

## Tech Stack
- Next.js + TypeScript
- Tailwind CSS
- Deployed on AWS (S3 + CloudFront)

## Algorithms
- Sorting: Bubble Sort, Merge Sort, Quick Sort (coming soon)
- Pathfinding: Dijkstra, A* (coming soon)

## Getting Started
npm install
npm run dev
